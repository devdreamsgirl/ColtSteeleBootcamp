Arrays come with many built in methods. These are the most common:

  • push/pop
  • shift/unshift
  • indexOf
  • slice

Push and Pop

The push method automatically adds a given value to the end of the array without having to call the index.

var colors = ["red", "orange", "yellow"];
      colors.push("green");  // 4 (returns length of the array.)
    colors;  //["red", "orange", "yellow", "green"]

The pop method automatically removes the last item in an array.

colors.pop();   // green (returns the removed item)
      colors;

Unshift and Shift

Shift and unshift work similarly to push and pop, but at the front of the array instead of the end.

colors.unshift("infared");  //4 (unshift returns array length and adds to the begining of the array)
        colors;   //["infared", "red", "orange", "yellow"]

      colors.shift(); // infared (shift removes the first item of the array and returns the removed item.)
    colors;   //["red", "orange", "yellow"]

with pop and shift, since they return the removed item, they can be stored easily in variables:

var col = colors.shift();    //"red"
var col = colors.pop();   //"yellow"

IndexOf

IndexOf finds the index of an item in an array.

var friends = ["Charlie", "Liz", "David", "Mattias", "Liz"];

//indexOf returns the FIRST index where a given item is found:
friends.indexOf("David");  //2
friends.indexOf("Liz");   //1 (not 4)

//returns -1 if item is not present:
friends.indexOf("Hagrid");   //-1

Slice

Use slice() to copy portions of an array.

var fruits = ["banana", "orange", "lemon", "apple", "mango"];
//use slice to copy the 2nd and 3rd fruits
//specify the index where the slice starts (inclusive) and ends (exclusive)
var citrus = fruits.slice(1,3);

//this does not alter the original fruits array.
//citrus contains ["orange", "lemon"]
//fruits contains ["banana", "orange", "lemon", "apple", "mango"]


//you can also use slice to copy an entire array.
var nums = [1, 2, 3];
var moreNums = nums.slice();

//both arrays contain [1, 2, 3]
