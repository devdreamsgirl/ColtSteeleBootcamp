Arrays are ways to hold groups of data together.

Previously, we could hold information by placing them in variables:

      var friend1 = "Charlie";
      var friend2 = "Liz";
      var friend3 = "David";
      var friend4 = "Mattias";

We could set each friend as a variable, but they would be individual and take a lot of code

It make more sense to use an array which holds the information as a group: 

      var friends = ["Charlie", "Liz", "David", "Mattias"];

The [] denotes an array and each value is separated by a comma. Arrays are indexed, begining with zero. i.e. Charlie index 0, Liz index 1, David index 2, Matias index 3. The indices are used to retreive data in the array.

      console.log(friends[0]) //"Charlie"
      friends[1] + " loves " + friends[2] // "Liz loves David"

We can also update the array values with the indices

      friends[0]= "Chuck";
      friends[1] = "Lizzie";

And you can add new data by accessing an index that has no value

      friends[4] = "Amelie";

There are two more ways to initialize arrays

var friends = []; //by naming an empy array
var friends = new Array(); //(uncommon) function to make an array (A must be capital)

Arrays can hole any type of data

var randomCollection = [49, true, "Herminone", null];

Arrays have a length property (begining with 1, instead of index 0)

var nums = [45, 37, 89, 24];
nums.length    //4
